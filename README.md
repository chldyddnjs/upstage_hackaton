# Installation
    1. git clone
    2. pip install -e .
    3. pip install requirements.txt
    
# preprocessing
    1. 법률 pdf 파일이 담긴 디렉토리를 준비한다.
    2. 전처리를 진행한다. 
        - python preprocess.pdf_to_json -r /path_to_your_file/
